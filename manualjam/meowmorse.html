On the Subject of Meowmorse

🐈 Is there a CAT indicator on the bomb?

  • This module looks like a Metamorse,but the hexagon button has triangle ears.Pressing the top right button plays a song with the cat's meow.This is a Morse code message of 3 to 6 letters. Decode Morse code according to the following rules:

    • If there are less than 2 indicators,take the sum of all letters in the serial number and subtract the number of "M","E","0(zero)" and "W",then modulo 3.Then go to 4th of list.
    • Otherwise,get the first three indicators of the alphabetical order.Take the first letter if it is lit and the third letter if it is unlit to make 3-letter words.If the word is "CAT",make the target-meow to middle tone and pass the next section.
    • Each letter of the created word, get the number on the alphabet,subtract the number of vowels in serial number and modulo 3.
    • If it is 0,the target-meow is low tone.If it is 1,the target-meow is middle tone.Otherwise,the target-meow is high tone.
    • Press the display to get the meow with middle tone.

  • Using the table below, use the results of the meow pitch compared to the target-meow as column and the meow length as row.

    • Table M30W5:

    lowersamehigher
    shortdotdashdot
    longdashdotdash

  • The word represents a name of the cat.Find the name in the below Table C4T5 and get the meow pitch of that cat and a phrase of the answer.
  • Press the arrow to set the pitch for transmission (the higher it looks up, the higher) then press the hexagon button to input the phrase into meow-morse code as well.Then press top-right button to submit phrase.
  • If a wrong word is submitted, it cause a strike and new song is selected.
  • A long press for more than 1 second is considered a DASH.
  • Whether the sound is higher or not is compared with the target-meow.
